In the late 1980's I was stationed at a Coast Guard Station in Frankfort Michigan. Multiple times a day we had Sailers and Hand Glider pilots call asking about the weather.
The data model for this application was generated from the Current Observation xsd of the National Weather Service
This model was chosen because it is simple enough for everyone to understand, complex enough to generate a good model, and provides a good chance for users to alter the code for their own preferences and location.

The data for the xmlPayload for the test was pulled from http://w1.weather.gov/xml/current_obs/KFKS.xml
